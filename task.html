What is html?

HTML stands for Hyper Text Markup Language A markup language is a text-encoding system which specifies the structure and formatting of a document HTML describes the structure of a Web page

what is tag? How many types of tags?

HTML tag is defined by a start tag and an end tag. HTML tags are broadly categorised into two main types: Paired (Container) Tags: These tags have both an opening tag and a closing tag. Unpaired (Empty/Self-closing) Tags: These tags do not have a closing tag; they are self-contained.

what is d/f b/w the tag and element?

A tag is just the markup that defines the start or end of an element. An element is the entire structure, including the tag, its content (if any), and its attributes.

what is the attribute?

Attributes provide additional information about elements Attributes are always specified in the start tag

href: Used for linking or referencing external resources. Commonly used in anchor and link elements. src: Used for embedding resources directly within the page. Commonly used in elements like , script and iframe

how many types of elements we have in html?

1. Block-level Elements 2. Inline Elements

what are d/f b/w block elements vs inline elements?

1. Block-level Elements These elements start on a new line and take up the full width available. Examples: div, p, h1 - h6, section, article, header, footer 2. Inline Elements These elements do not start on a new line and only take up as much width as necessary. Examples: span, a, img, strong, em, label

what are semantic vs non-semantic elements?

Semantic elements clearly communicate the role and structure of the content to browsers and developers. Non-semantic elements are generic and do not provide information about the content's meaning or structure. They are primarily used for layout and styling purposes.

d/f b/w div and span?

div is a block-level element used to structure larger sections of a webpage, affecting the layout and breaking content into blocks. span is an inline element used for styling or grouping smaller pieces of content without affecting the overall layout.

table vs li ?

table: Used to display data in a structured, grid-like format with rows and columns. li: Used to define items in a list (either ordered or unordered), suitable for simpler, list-style content.

what are meta tags?

Meta tags are HTML elements that provide metadata (information about the webpage) to browsers and search engines.

what is the use of forms in html?

Forms in HTML are used to collect user input and send it to a server for processing.

what are the form elements?

form: The container for the form elements. input: Accepts various types of input like text, passwords, numbers, or files. textarea: Allows multi-line text input. select: Creates a drop-down menu for user choices. button: Submits the form data to the server. label: Describes the purpose of the form elements for accessibility.

what are the iframes?

what is the advantage of collapsing in whitespace?

Collapsing whitespace simplifies HTML coding, enhances readability, ensures consistent rendering, and reduces file size, making it easier for developers to manage and maintain web pages.

what are the html entities?

HTML entities are essential for including special characters and reserved symbols

what are the features of html 5?

HTML5 significantly enhances the capabilities of web applications by introducing new semantic elements, multimedia support, improved forms, and various APIs.

are the elements and tags are the same thing in html?

Tags are the individual markup components used to create elements, while elements are the full structures formed by those tags and their content.

What is git???

Git is a version control system. Git helps you keep track of code changes

what is d/f b/w git vs github?

Git is the version control system that manages code and changes locally GitHub is a collaborative platform that hosts Git repositories online and adds tools for teamwork and project management.

What is the local repo and github repo?

A local repository is on your computer, enabling offline development and independent work GitHub repository is a remote repository on GitHub that facilitates collaboration and sharing among multiple developers.

what is the d/f b/w the push and pull in git???

Push: Uploads local changes to a remote repository. Pull: Retrieves and integrates changes from a remote repository into the local repository.

what are the ways of getting code from github?

Clone for a local development environment with full Git functionality. Download ZIP for a quick access to the code without version control. Use GitHub Desktop for a GUI-based approach. Fork a repository to create your own copy for modifications. Access the GitHub API for programmatic interactions.

what is the flow of git???

Working Directory: Where you make changes. Staging Area: Where you prepare changes for commit. Local Repository: Where you save committed changes. Remote Repository: Where you push changes to share with others
